Trabalho intermediário de Java OO

Escreva um programa que:
	- Tenha no mínimo 2 clases distintas;
	- Ao menos uma das classes deverá possuir 3 métodos públicos que realizem uma 
	ação relacionada ao tema;
	- Defina os atributos das classes e crie encapsulamento para eles;
	- Crie ao menos um construtor de classe com parâmetro;
	- Deverá ter uma classe principal com o método main para testar seu programa;
	- Use Sysout para imprimir os dados do seu programa no console;
	- Você deverá descrever o que seu programa faz em um breve texto com o passo a passo;
	- Crie as tabelas no Figma ou em outro programa para definir o relacionamento das classes,
	seus métodos e atributos (https://www.figma.com/login), https://app.diagrams.net/
	
	Seu trabalho final deverá ter 4 itens principais:
	1 - As definições documentada do que seu programa deve fazer
	2 - O programa em Java
	3 - O diagrama de classes
	4 - O texto explicando o passo a passo
